=============================================================================
翻訳ファイルについて / About Translation Files
=============================================================================

このディレクトリには以下の翻訳ファイルが含まれています：

dialog.tra (100,267行)
  - ゲーム内の全ての文字列の日本語翻訳
  - 男性キャラクター、または性別に依存しないテキスト用
  - 文字列番号: @0 ~ @71403

dialogf.tra (100,254行)
  - ゲーム内の全ての文字列の日本語翻訳（女性版）
  - 女性プレイヤーキャラクター用の翻訳
  - 性別によって表現が異なるテキストに使用

setup.tra
  - インストーラーUIの翻訳
  - コンポーネント名、メッセージなど

game.tra
  - 追加のゲーム内テキスト
  - Mod固有の文字列

=============================================================================
WeiDUでの使用方法
=============================================================================

これらの.traファイルはWeiDUの LANGUAGE 宣言に含まれており、
インストール時に自動的に以下のように処理されます：

1. WeiDUがtraファイルを読み込む
2. 日本語版のdialog.tlkファイルを生成
3. ゲームの言語/ja_JP/フォルダに配置
4. ゲームの設定を日本語に変更

プレイヤーはゲーム起動後、設定で「Japanese (日本語)」を
選択することで、翻訳されたテキストを使用できます。

=============================================================================
文字列の形式
=============================================================================

@番号 = ~翻訳テキスト~

例:
@1 = ~何故に私の邪魔をしに来たのだ？礼儀を知らぬのか？出て行け！~
@100 = ~ファイター~

特殊なタグ:
<CHARNAME> - プレイヤーキャラクターの名前
<PRO_~> - 代名詞（彼/彼女など）
[音声ファイル名] - 音声ファイルへの参照
