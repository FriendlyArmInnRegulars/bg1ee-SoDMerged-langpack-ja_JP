===============================================================================
BGEE & SoD Japanese Language Pack
===============================================================================

Version: v1.0.0
Author: FriendlyArmInnRegulars
Game: Baldur's Gate Enhanced Edition + Siege of Dragonspear

===============================================================================
Overview
===============================================================================

This is a WeiDU mod that provides Japanese localization for Baldur's Gate
Enhanced Edition (BGEE) and the Siege of Dragonspear (SoD) expansion.

It translates in-game text, dialogues, and UI elements into Japanese.

===============================================================================
Requirements
===============================================================================

- Baldur's Gate Enhanced Edition (Steam/GOG/Beamdog)
- Siege of Dragonspear expansion (optional, required for some components)
- WeiDU (included with installer)

===============================================================================
Installation
===============================================================================

1. Locate your game installation folder
   - Steam: Steam\steamapps\common\Baldur's Gate Enhanced Edition
   - GOG: GOG Games\Baldur's Gate Enhanced Edition

2. Copy this mod folder to your game directory

3. Run the installer
   - Windows: Double-click setup-bg1ee_sod_jp.exe
   - Mac/Linux: Run in terminal:
     weidu setup-bg1ee_sod_jp.tp2 --log setup-bg1ee_sod_jp.debug

4. Select components to install
   [0] BGEE & SoD Japanese Translation (required)
   [10] Siege of Dragonspear Content Translation (for SoD owners)
   [20] UI Elements Translation (recommended)

5. Launch the game to verify Japanese localization

===============================================================================
Uninstallation
===============================================================================

1. Run setup-bg1ee_sod_jp.exe (or weidu) again
2. Select 'U' to uninstall
3. Or manually restore from bg1ee_sod_jp/backup folder

===============================================================================
Known Issues
===============================================================================

- Some fonts may not display Japanese characters correctly
- Long Japanese text may overflow UI elements
- May conflict with other language mods

===============================================================================
Compatibility
===============================================================================

Compatible with:
- BG:EE v2.6 and later
- SoD expansion

May conflict with:
- Other language pack mods
- Mods that modify dialog.tlk
